# Claude Architect Foundations Trainer

Claude Certified Architect - Foundations を目指す日本語話者向けの学習ツールです。
公式問題の再掲ではなく、公開可能な自作シナリオ、重要語、判断軸、実装練習で知識を固める構成です。

## 使い方

`index.html` をブラウザで開くと使えます。ローカルサーバーで開く場合は、このフォルダで次を実行します。

```powershell
python -m http.server 8765 --bind 127.0.0.1
```

その後、ブラウザで `http://127.0.0.1:8765/index.html` を開いてください。

## 学習の流れ

1. 「自作シナリオ特訓」で英文の問題文を読む
2. 選択肢を見る前に「何が問われているか」を日本語で書く
3. 分からない英単語を「英単語ゼロ残し」で潰す
4. 解説で判断軸を確認する
5. 「英文読解体力」で翻訳なしの読解を短時間くり返す

進捗はブラウザのローカル保存に記録されます。

## GitHub Pagesで公開する

スマホからURLで使うなら、GitHub Pagesで公開するのが手軽です。

1. GitHubで新しいリポジトリを作る
2. このフォルダのファイル一式をリポジトリのルートへアップロードする
3. リポジトリの `Settings` → `Pages` を開く
4. `Build and deployment` の `Source` を `Deploy from a branch` にする
5. `Branch` を `main`、フォルダを `/root` にして保存する
6. 数分後に表示されるURLをスマホで開く

スマホでは、ブラウザの共有メニューからホーム画面に追加できます。一度開いた後は、基本ファイルが端末にキャッシュされます。
